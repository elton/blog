使用gorun来用Go撰写脚本

Go也可以用来写脚本，类似shell

使用go get安装 

go get launchpad.net/gorun

如果提示 package launchpad.net/gorun: exec: “bzr”: executable file not found in $PATH 说明你缺少bazaar版本控制工具，launchpad都是用bazaar的。

如果你是mac，需要执行下面的命令(需要事先安装brew） 

brew install bazaar
如果是ubuntu，需要执行下面命令： 
sudo apt-get install bzr
